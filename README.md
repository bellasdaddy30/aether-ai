# AETHER — Premium AI Companion

A full-stack AI chat application with voice, image generation, WebXR VR, and Stripe-integrated subscription tiers. Built to run on low-resource hardware (2-core CPU, 4GB RAM).

---

## Tech Stack

| Layer | Technology | Why |
|---|---|---|
| Server | Fastify (Node.js) | 3–4× faster than Express, tiny footprint |
| Database | SQLite (better-sqlite3) | Zero config, synchronous, RAM-friendly |
| LLM | OpenRouter | Access multiple models, pay-as-you-go |
| Images | Replicate | No upfront cost, SDXL + Flux |
| Voice | ElevenLabs TTS + Web Speech API STT | Free STT, 10k chars/mo free TTS |
| Payments | Stripe | No monthly fee, % per transaction |
| Frontend | Vanilla JS + CSS | No bundler = zero dev-server RAM cost |

## Prerequisites

- Lubuntu 22.04+ (or any Ubuntu-based distro)
- Node.js 18+ (script installs it)
- An internet connection for API calls

## Quick Start (Lubuntu Terminal)

```bash
# 1. Clone or copy the project
cd ~
# (copy the aether/ folder here)
cd aether

# 2. Run the setup script
chmod +x scripts/setup.sh
bash scripts/setup.sh

# 3. Edit your API keys
nano .env

# 4. Inject Stripe price IDs into frontend HTML
node scripts/inject-prices.js

# 5. Start the server
npm start

# Open http://localhost:3000
```

## Setting Up Stripe (Step by Step)

1. Sign up at https://stripe.com (no monthly fee)
2. Go to **Products** → **Add Product**
3. Create three products:
   - **AETHER Basics** — $9.99/month recurring
   - **AETHER Elite** — $24.99/month recurring
   - **AETHER God-Mode** — $49.99/month recurring
4. Copy each **Price ID** (starts with `price_`) into `.env`
5. Run `node scripts/inject-prices.js` to embed them in the frontend
6. For local webhook testing, install the Stripe CLI:
   ```bash
   # Download from https://github.com/stripe/stripe-cli/releases
   # Then:
   stripe login
   stripe listen --forward-to localhost:3000/api/stripe/webhook
   # Copy the webhook secret printed to STRIPE_WEBHOOK_SECRET in .env
   ```

## OpenRouter Model Setup

1. Sign up at https://openrouter.ai
2. Go to **Keys** → create an API key
3. Free models available (e.g. `mistralai/mistral-7b-instruct:free`)
4. Paid models charge per token — no subscription required
5. Set your desired models in `.env`:
   ```
   MODEL_BASICS=mistralai/mistral-7b-instruct:free
   MODEL_ELITE=nousresearch/hermes-3-llama-3.1-70b
   MODEL_GODMODE=anthropic/claude-sonnet-4-5
   ```

## RAM-Saving Tips for Lubuntu

### Reduce Node.js memory usage (already in npm start)
```bash
node --max-old-space-size=512 --optimize-for-size server/index.js
```

### Add swap space (strongly recommended)
```bash
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```

### Reduce swappiness (use RAM before swap)
```bash
echo 10 | sudo tee /proc/sys/vm/swappiness
```

### Kill memory hogs before dev work
```bash
# Find memory-heavy processes
ps aux --sort=-%mem | head -10

# Kill unnecessary background apps
pkill -f thunderbird  # if not needed
pkill -f libreoffice  # if not needed
```

### Use a lightweight terminal
LXTerminal (default on Lubuntu) is already lightweight.

### Monitor RAM during development
```bash
# In a separate terminal:
watch -n 2 free -h
```

## Deployment (Free Options)

### Railway.app
```bash
# Install Railway CLI
npm install -g @railway/cli
railway login
railway init
railway up
# Set env vars in Railway dashboard
```

### Render.com
- Connect GitHub repo
- Set build command: `npm install`
- Set start command: `npm start`
- Add environment variables in dashboard
- Free tier: 512MB RAM (sufficient for SQLite + Fastify)

### Fly.io
```bash
npm install -g flyctl
fly auth login
fly launch
fly secrets set OPENROUTER_API_KEY=xxx JWT_SECRET=xxx ...
fly deploy
```

## iPhone Access

Once deployed or running on your local network:
1. Find your laptop IP: `ip addr show | grep 'inet '`
2. Set `HOST=0.0.0.0` in `.env` (already default)
3. Open `http://YOUR_LAPTOP_IP:3000` in Safari on iPhone
4. Voice input (STT) works natively in Safari via Web Speech API
5. Add to Home Screen for a full-screen PWA experience
